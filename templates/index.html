📚 About This Project

🗃️

Dataset — MNIST

70,000 handwritten digit images. 60,000 train + 10,000 test. 28×28 grayscale, 10 classes (0–9).

🧠

CNN Architecture

2 Conv blocks (32→64 filters) with BatchNorm, ReLU, MaxPool, Dropout. 3-layer FC classifier.

🎯

Performance

~99.2% test accuracy. Dropout (0.25 / 0.5 / 0.3) prevents overfitting. Early stopping used.

⚙️

Training Setup

Adam (lr=1e-3, wd=1e-4), CrossEntropy loss, ReduceLROnPlateau, early stopping patience=7.
